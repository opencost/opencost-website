You may check out the source for the [OpenCost Helm Chart](https://github.com/opencost/opencost-helm-chart/) or you may install the Helm chart directly to your Kubernetes cluster. Review the [values.yaml](https://github.com/opencost/opencost-helm-chart/blob/main/charts/opencost/values.yaml) for the settings available for customization. With your custom settings in the Helm values file `local.yaml`, install OpenCost:

```sh
helm install opencost --repo https://opencost.github.io/opencost-helm-chart opencost \
  --namespace opencost -f local.yaml
```
